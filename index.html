Principais Funcionalidades

  • git clone : Este comando realiza a cópia do reposítório remoto para sua máquina local.
  • git pull: Este comando realiza a atualização do seu local com o repositório remoto.
  • git status: Este comando verifica se algum arquivo foi modificado/inserido no projeto.
  • git checkout -b : Este comando cria uma branch de trabalho local.
  • git add : Este comando adiciona os arquivos que foram modificados/inseridos na sua branch.
  • git commit -m "mensagem": Este comando insere uma mensagem ao seu commit para identificar as alterações.
  • git push origin : Este comando realiza o envio das suas alterações da máquina local para o repositório remoto.
